# FAQ

## GraphLens не видит текст моей коллекции
Проверьте `GET /api/qdrant/collections/{name}/probe`. Если `has_text=false` -
в payload нет текста чанков (некоторые RAG хранят текст отдельно, например в
PostgreSQL). GraphLens работает только с данными Qdrant.

## Какие модели подходят для извлечения?
Любые с OpenAI-совместимым API (LM Studio, vLLM, Ollama, облачные).
Рекомендуется модель с хорошим следованием формату JSON и контекстом 8k+.

## Сколько стоит извлечение?
Один LLM-запрос на батч из `EXTRACT_BATCH_SIZE` чанков (по умолчанию 5).
70K чанков = 14K запросов - планируйте локальный инференс.

## Как очистить граф одной коллекции?
`DELETE /api/graph?collection=name` или кнопка «Очистить граф» в UI.
