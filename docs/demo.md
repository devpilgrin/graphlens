# Демо: от нуля до графа за 5 минут

## 1. Поднимите Neo4j
```bash
docker compose up -d
```

## 2. Загрузите демо-корпус
```bash
python scripts/seed_demo.py ../path/to/your/document.md --collection lens_demo
```

## 3. Запустите бэкенд и фронтенд
```bash
bash scripts/dev.sh
```

## 4. Постройте граф
Откройте http://localhost:3300/?collection=lens_demo, выберите `lens_demo`,
нажмите «Построить граф». Прогресс - в панели «Извлечение графа».

## 5. Поищите
Пример: «какие тарифы и квоты у платформы?» - найденные сущности чипсами,
карточки чанков с бейджем «через граф», клик по сущности центрирует канвас.
